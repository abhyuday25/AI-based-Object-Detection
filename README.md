AI-based object detection is a full-stack AI-powered web application designed to address the global
challenge of food waste by automating grocery item identification, quantity estimation, and
freshness prediction using Google's Gemini 2.0 Flash multimodal AI. This assessment
evaluates the project's strategic positioning through a comprehensive SWOT analysis,
examines financial and human resources, describes the organisational structure, and
proposes strategic recommendations to ensure long-term sustainability and market
viability.
